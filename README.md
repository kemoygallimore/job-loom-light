# RizonHire

Multi-tenant Applicant Tracking System with built-in video screening, public careers portals, and a super-admin platform layer.

## Documentation

- [Feature inventory](./docs/01-features.md)
- [Technical documentation](./docs/02-technical.md)
- [User guide](./docs/03-user-guide.md)

## Stack

React 18 · Vite 5 · TypeScript · Tailwind · shadcn/ui · TanStack Query · Supabase (Postgres + Auth + Edge Functions) · Cloudflare R2.

## Local development

```bash
npm install
npm run dev
```
